# Machine Learning Empowered Precision in Cancer Image Classification

This repository contains the implementation of my MSc research project focused on applying **machine learning and deep learning techniques for breast cancer detection using histopathology images**.

The project investigates how convolutional neural networks can assist in medical diagnosis by classifying cancerous and non-cancerous tissue images.

---

## Project Overview

Breast cancer remains one of the leading causes of death among women worldwide. Early detection plays a critical role in improving survival rates. This project explores the use of **deep learning architectures to automate breast cancer image classification**.

The study evaluates different deep learning approaches to improve classification performance and support medical decision-making.

---

## Research Objective

The objective of this research is to:

- Apply deep learning techniques for cancer image classification
- Evaluate CNN architectures for medical image analysis
- Improve classification accuracy for breast cancer detection
- Demonstrate the potential of AI in healthcare diagnostics

---

## Model Architecture

The implementation focuses on **Convolutional Neural Networks (CNN)**, particularly:

- **VGG16 architecture**
- Transfer learning for feature extraction
- Fully connected layers for binary classification
