# Jupyter Notebooks

This folder contains the Jupyter notebooks used in the **Breast Cancer Classification** project.

The notebooks demonstrate the implementation and experimentation of machine learning and deep learning models for detecting breast cancer from histopathology images.

---

## Notebook

### machine-learning-cancer-detection.ipynb

This notebook contains the implementation of the breast cancer classification model.

Main components included:

- Data loading and preprocessing
- Image preparation and dataset balancing
- Training and testing split
- Deep learning model implementation using **VGG16 architecture**
- Model training and evaluation
- Performance analysis using classification metrics

---

## Technologies Used

The notebook uses the following libraries:

- Python
- NumPy
- Pandas
- OpenCV
- TensorFlow / Keras
- Scikit-learn
- Matplotlib
- Seaborn

---

## Purpose of the Notebook

The goal of this notebook is to demonstrate how **deep learning techniques can assist in medical image classification**, specifically for detecting breast cancer.

The notebook serves as the **experimental implementation** supporting the research project:

**"Machine Learning Empowered Precision in Cancer Image Classification"**
